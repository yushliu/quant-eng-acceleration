# Portfolio Definition

## Baseline Portfolio

- Tickers: SPY, QQQ, TLT, GLD
- Weights: equal weight
- Purpose: balanced starting point for the first implementation pass

## Comparison Portfolio

- Tickers: SPY, QQQ, IWM, TLT
- Weights: equal weight
- Purpose: more equity-heavy comparison to test whether model behavior stays consistent across portfolio composition changes

## Design Rationale

The two-portfolio setup was chosen to test whether model behavior remains consistent across a more balanced allocation and a more equity-heavy allocation.
