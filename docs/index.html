About

Quantitative Engineering Acceleration

Quantitative Engineering Acceleration is a student-run club focused on reproducible benchmarking. We study CPU, GPU, and FPGA simulation and share results with clear, versioned artifacts.

We are a hands-on group that builds repeatable experiment pipelines and compares methods fairly. Our goal is to make performance claims traceable, reviewable, and easy to reproduce.

What We Do

  • Use a clear workflow for experiments.
  • Record methods and environments.
  • Share results with version tags.
  • Review changes on a regular schedule.

Reproducibility

Write down the following for each run:

  • Hardware details.
  • Software versions.
  • Build flags and settings.
  • Input data and parameters.
  • Runtime and output checks.

Results And Releases

We use tags and releases to mark results. Each release should include notes and links to data. Store large data outside Git if needed.
