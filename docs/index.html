Featured Project

Risk Model Comparison

This project builds a quantitative benchmark for comparing Historical Simulation, Parametric Normal, and EWMA + Monte Carlo VaR/CVaR models under shared data, portfolio, and evaluation rules, creating a clearer basis for model selection and future club releases.

Open Project

Project Role

Shared model-selection benchmark for future quantitative releases.

Why It Matters

Makes risk-model tradeoffs more interpretable by keeping data, portfolio setup, and evaluation logic aligned.

Why it stands out

It reflects the club’s benchmark-driven approach: models are compared under controlled assumptions, shared evaluation rules, and reusable reporting structure.

How We Work

The club uses a consistent method so implementation work, comparison logic, and published outputs remain connected.

Build with controlled assumptions

We define scope, setup, and comparison rules before implementation so later outcomes stay interpretable.

Compare with traceability

We evaluate methods under shared data, parameters, and reporting logic so differences can be reviewed rather than guessed.

Publish for reuse

We package notes, artifacts, and structured summaries so work can be revisited, extended, and reused across later projects.
